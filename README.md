# 简介

魔改了这个项目，增加了fofa聚合api调用以及其他优化

https://github.com/wgpsec/fofa-search

# 修改的地方

- 使用fofa聚合api调用该hostname的指纹数据
- amap.fofa.info数据调用在backgroud.js允许并且写入缓存，优化加载
- domain，host，port增加点击跳转以及fofa查询
